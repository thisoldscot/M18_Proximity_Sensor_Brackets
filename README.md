# M18 Proximity Sensor Brackets

[![CC BY-NC-SA 4.0][cc-by-nc-sa-shield]][cc-by-nc-sa]

## About the Project

| ![M18 Proximity Sensor Brackets](https://github.com/thisoldscot/M18_Proximity_Sensor_Brackets/raw/main/Render/M18_Proximity_Sensor_Brackets_2017-Dec-04_04-08-37PM-000_FRONT_jpg.jpg) |
| :----------------------------------------------------------------------------------------: |

Straight and right-angled mounting brackets for M18 barrel proximity sensors, with three
spacer depths — 10&nbsp;mm, 15&nbsp;mm and 20&nbsp;mm — so you can set the sensor stand-off
to suit the job.
