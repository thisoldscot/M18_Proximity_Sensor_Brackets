# M18 Proximity Sensor Brackets

Straight and right-angled mounting brackets for M18 barrel proximity sensors, with three
spacer depths — 10&nbsp;mm, 15&nbsp;mm and 20&nbsp;mm — so you can set the sensor stand-off
to suit the job.
